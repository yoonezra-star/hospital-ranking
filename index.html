자주 묻는 질문

병원찾기 데이터 기준과 운영 원칙을 빠르게 확인하세요

병원 정보는 어디에서 오나요?

건강보험심사평가원과 공공데이터포털 공개 API, 공개 가능한 병원 안내 정보를 바탕으로 정리합니다.

병원 순서는 광고 순서인가요?

아닙니다. 광고와 별도로 평점, 후기 수, 전문의 수, 최신성 등 내부 기준을 반영해 병원 목록을 정리합니다.

공공 API가 느리면 어떻게 되나요?

실시간 공공 API 응답이 늦을 때는 상태 배지를 표시하고, 보유 중인 보강 데이터와 요약 정보를 우선 제공합니다.

병원 정보 수정 요청은 어떻게 하나요?

병원명, 수정 내용, 근거 정보를 정리해 문의하기 또는 replyleaders@naver.com으로 보내주시면 검토 후 반영합니다.
